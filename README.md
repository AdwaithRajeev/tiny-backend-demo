# Tiny Backend Demo

A simple login and registration system built using Node.js, Express, and MySQL.

## Features
- User registration
- User login with database validation
- Redirect to home page after successful login
- Basic backend routing using Express

## Tech Stack
- HTML
- JavaScript
- Node.js
- Express.js
- MySQL

## How to Run
1. Clone the repository
2. Run `npm install`
3. Start MySQL and create the database
4. Run `node index.js`
5. Open `http://localhost:3000`
